  Below is the inventory of available functions and the operational guide for deployment.

  5. Deployment & Operational Workflow

  To deploy this system into a weekly routine (e.g., "Sunday Night Review"), follow this sequence:

  Step 1: Refresh Market Data
  Fetch the latest positions and transactions from Schwab and update the Google Sheet.

   1 python manager.py snapshot --source schwab --live
   2 python manager.py sync transactions --days 30 --live

  Step 2: Update the Vault
  If you have written new theses in vault/theses/, freeze them into the system.
   1 python manager.py vault snapshot

  Step 3: Build the AI Context
  Create the composite bundle that the agents will use.
   1 python manager.py bundle composite

  Step 4: Run Full Analysis
  Execute all agents and write their findings to the Agent_Outputs tab for the Streamlit UI to display.

   1 python manager.py analyze-all --live

  a single command. You can run it via python scripts/live_update.py.
