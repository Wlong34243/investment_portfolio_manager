## Deployment Rules

<!-- Institutional discipline: write these now so future-you can't revise in panic. -->

1. **Small steps.** Starter sizes as specified per entry. No going big on any single name in one move.
2. **No removing names from the list during a drawdown** except on actual thesis break (new information, not new emotion).
3. **Adding names during a drawdown is allowed** if the name clearly fits one of Bill's four styles and can get a written thesis.
4. **Tranches for every entry.** Initial + add + final. Never deploy full size in one trade.
5. **Re-buys, adds, and new ideas are tracked separately** to prevent the list from becoming dominated by whichever feels easiest in the moment.
