# INTERFACE MAP: METRIC DERIVATIONS

| UI Element | Source / Metric | Derived From |
|------------|-----------------|--------------|
| Total Value | Holdings_Current | SUM('Market Value') |
| Portfolio Beta | Risk Engine | Weighted average of (Ticker Beta * Weight) |
| Blended Yield | Income Engine | (Annual Income / Total Invested Value) * 100 |
| Realized G/L | Realized_GL | SUM('Gain/Loss') in the Realized_GL tab |
| Macro Triggers | Macro Monitor | Live API (St. Louis Fed / Yahoo Finance) |
| RE Equity | Net Worth Page | (RE Valuation @ 6% Cap) - Total Debt |
| Concentration | Risk Tab | Logic: Any position > 10% weight (Warning) |
| Drift Alert | Rebalancing | (Current Weight - Target Weight) > Threshold |
| Total Net Worth | Net Worth Page | (Liquid Assets) + (RE Equity) + (RE Reserves) |

## LOGIC & DERIVATIONS:
- **Real Estate Valuation:** (Annual NOI / 0.06 Cap Rate). Fallback: 'property_value' from RE Sheet.
- **RE Equity:** (RE Valuation) - (Total Debt).
- **Liquid Assets:** SUM(Market Value) from current holdings.
- **Data Source (RE):** Dashboard ID `1DXuY1iBo2GqZCCSZ7OrUa4iaunb5s8Kf1Rms8Z237rQ`.
- **Target RE Cells:** B21 (Debt), B20 (Debt Service).
