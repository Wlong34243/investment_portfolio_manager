1. System Architecture Overview

The system is a Headless CLI application designed for high-integrity financial analysis. It follows a strictly decoupled architecture:

  • Data Ingestion: Pulls from Schwab API or CSV.
  • The "Frozen State" Pattern: Every run captures a Context Bundle (JSON) containing all market and research data. This ensures the AI never hallucinates numbers based on stale or shifting data.
  • Intelligence Layer: Specialized Python agents send curated "Context Bundles" to Gemini 2.5 Pro with structured Pydantic schemas.
  • View Layer: Results are written back to Google Sheets via gspread.
Philosophy: Dry Run by Default. No script will write to your Google Sheets unless you explicitly pass the --live flag. This prevents accidental data corruption and unnecessary API costs.

2. Core Engine

The core engine consists of the orchestration layer and the data spine.

File Role Key Responsibilities
manager.py CLI Entry Point Orchestrates snapshot, analyze-all, vault, and dashboard commands.
pipeline.py Data Spine Handles "Nuclear Type Enforcement," normalization, and core sheet writing logic for Holdings and Snapshots.
config.py Settings Single source of truth for Tab names, Sheet IDs, API keys, and Ticker constants.
core/bundle.py State Freezer Builds the Market Bundle by joining positions with yfinance/Schwab quotes.
core/vault_bundle.py Research Freezer Packages local markdown theses and transcripts into a JSON bundle.

Manager Commands

python manager.py snapshot --live          # Sync Schwab and update Holdings
python manager.py analyze-all --live       # Run all AI agents and update Dashboard
python manager.py dashboard refresh --live  # Re-apply formatting and rebuild Action Views
python manager.py vault snapshot           # Freeze research state

6. Agent Tuning Guide (/agents)

Intelligence is driven by the System Prompts. The behavior of any agent (Valuation, Tax, Macro) is dictated by plain English text files, not Python code.

How to Change Agent Logic:
  1. Locate the prompt in agents/prompts/[agent_name]_system.txt.
  2. Edit the text (e.g., "Always prioritize high-dividend stocks" or "Be more aggressive with TLH").
  3. Run python manager.py analyze-all --live. The agent will immediately use the new rules.
